## Responsive masonry component for React.js


### Install

Install using `npm install react-grid-masonry`

### Usage 

In a React app, use the Button or Badge components:  
`import { Masonry } from 'react-grid-masonry'`  

#### Masonry props

| Name        | Description      
| ----------- | -----------      
| comp        | React Component
| items       | Array          
| columnWidth | Number         
| minCols     | Number         
| gutter      | Number
